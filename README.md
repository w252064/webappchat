
###  **项目介绍** 
项目大概完成了商城后台，小程序所有页面和功能，wap的部分功能

面向学习型的开源框架，简洁高效，减少过渡封装，展现技术本质

Springboot作为容器，使用mybatis作为持久层框架

使用官方推荐的thymeleaf做为模板引擎，shiro作为安全框架,主流技术，“一网打尽”

基于注解的sql写法，零XML，极简配置，一键前后台代码生成



###  **功能简介**
